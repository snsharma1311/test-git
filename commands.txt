1. git init
to add empty git

2. git remote add origin "<url to github proj repo>"

3. git pull origin master
to pull all the files in to the local machine

4. git status
to check changes files etc

5. git add <file>
to add file to the index

6. git add -A 
to add multiple files/folder

7. git commit -m "message"

8. git commit -a -m "message"
to commit multiple files

9. git branch <branch_name>

10. git checkout <branch_name>
work on branch

11. git checkout <des branch name>; git merge <branch to be merged with des branch>
merge the souce branch to destination branch

12. ssh-keygen
generate public key for github account

13. setting->add ssh-><copy paste key>

14. git push origin <branch_name>
